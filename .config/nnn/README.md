## This version of nnn includes Nerd Fonts icons and Git signs.

### Git signs

| Sign     | Meaning |
| -------- | ------- |
| +        | Add     |
| - (bold) | Delete  |
| !        | Ignore  |
| pencil   | Modify  |
| ?        | New     |
| -        | Non     |
| clock    | Update  |

### Compile:

```
sudo make O_NERD=1 O_GITSTATUS=1 install
```
